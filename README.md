# 📚 सुमित कुमार गुप्त की डिजिटल पुस्तकमाला

विद्यालयीन जीवन की स्मृतियों, अनुभवों और संस्मरणों पर आधारित पुस्तकों का एक डिजिटल संग्रह।

> * ❝ पुस्तकें केवल शुष्क जानकारियों अथवा ज्ञानार्जन का संकलन मात्र नहीं हैं,
अपितु वे जीवन के उन अनमोल, जीवंत और चंचल क्षणों को सदैव के लिए
स्मृतियों के झरोखे में सुरक्षित रखने का माध्यम हैं, जो अनजाने में ही
हमें परिपक्वता प्रदान करते हैं और श्रेष्ठ मानव बनाते हैं। ❞*

👤 **लेखक:** सुमित कुमार गुप्त
